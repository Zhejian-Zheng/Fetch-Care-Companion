# Fetch Care Companion Mobile

React Native / Expo version of the Fetch Care Companion concept.

## Run

```bash
cd mobile
npm install
npm run start
```

Open it with Expo Go, iOS Simulator, Android Emulator, or Expo web.

## What it shows

- Mobile-first care dashboard
- Pet selection
- Visit type selection
- Async vet estimate reader
- Coverage result, line items, missing documents, and direct-to-vet claim timeline
